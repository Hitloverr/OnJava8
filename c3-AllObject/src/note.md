# 数据存储
1. 寄存器：CPU内部，存储区域最快，但是程序员不感知，不能操纵。
2. 栈内存：内存RAM中，移动栈顶指针分配与释放内存
3. 堆内存：所有java对象都存在这里
4. 常量存储：字符串常量池：在静态存储中
5. 非RAM存储：序列化对象| 持久化对象。
